## Electoral Transparency Commitment

VectisDev and the VIGIL project commit to:

1. **Non-partisan operation** — The system monitors mathematical consistency of
   published CNE data. It does not favour any candidate, party, or outcome.

2. **Public auditability** — All detection rules, thresholds, and anomaly
   definitions are documented and open to peer review.

3. **Data integrity, not alteration** — VIGIL reads and hashes public CNE
   data. It never modifies, intercepts, or delays the official count.

4. **No personal data** — The system does not collect, store, or transmit
   personally identifiable information about voters or officials.

5. **Reproducible builds** — The GitHub Pages deployment is fully automated
   and traceable via the `pages.yml` workflow. Any observer can verify
   that the deployed site matches the source code.

6. **Graceful shutdown** — If the CNE ceases publication, the system enters
   safe mode and alerts operators without generating false anomalies.
