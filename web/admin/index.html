§6 Manual
Guía rápida del panel de administración

Modo Fácil vs Modo Completo

Modo Fácil expone los 4 parámetros más importantes: intervalo de captura, requests/hora, umbrales Benford y Z-score, y el topic de ntfy. Ideal para operación diaria.

Modo Completo habilita 5 tabs con acceso a todos los parámetros: endpoints individuales, watchdog, rate limiter, red, y reglas de detección.

El mapa de endpoints

Muestra los 18 departamentos + NACIONAL de Honduras como puntos geoposicionados. El color indica el estado del endpoint configurado. Click en cualquier punto para editar la URL o probar el endpoint.

Los cambios desde el popup quedan pendientes (dirty) hasta que hagas clic en "Aplicar cambios".

Cómo se guardan los cambios

1. Modifica cualquier control — el botón "Aplicar cambios" se activa.
2. Click en "Aplicar" → modal con diff de todos los cambios pendientes.
3. Confirma → se te pide el GitHub PAT (solo en sessionStorage, desaparece al cerrar el browser).
4. El sistema hace commit a config/prod/ vía GitHub API.
5. Los cambios toman efecto en el próximo ciclo (~15 min).

Animal mode

NORMAL: operación estándar.
CAUTION: el sistema detectó condiciones adversas, reduce agresividad.
SURVIVAL: modo mínimo, solo captura esencial, máxima resiliencia.

Qué hacer si un endpoint queda rojo

1. Click en el punto en el mapa → "Probar ahora" para verificar si responde.
2. Si no responde: verifica la URL con el proveedor CNE.
3. Actualiza la URL en el popup → "Guardar cambio" → "Aplicar cambios".
4. Si es un fallo temporal, el circuit breaker se recupera automáticamente.

Recuperar configuración anterior

Todo cambio es un commit en config/prod/. Para revertir: ve a GitHub → config/prod/endpoints.yaml → History → elige el commit anterior → "Revert" o copia el contenido manualmente.

GitHub PAT requerido

Para aplicar cambios en el repositorio se necesita un Personal Access Token (Classic) con scope repo. Se guarda solo en sessionStorage y desaparece al cerrar el browser.

Cambios pendientes

Los cambios toman efecto en el próximo ciclo de captura (~15 min).
