¿Qué es OTS y cómo verificar?What is OTS and how to verify?

Cada checkpoint tiene un Merkle root (huella digital). OTS lo envía a 4 calendarios públicos que lo incluyen en una transacción Bitcoin. Cualquier auditor puede verificar que el dato existía antes de cierto bloque sin depender de VIGIL.Each checkpoint has a Merkle root (digital fingerprint). OTS sends it to 4 public calendars that include it in a Bitcoin transaction. Any auditor can verify the data existed before a certain block without relying on VIGIL.

Calendarios usados (gratuitos, sin API key):Calendars used (free, no API key):

  • bob.btc.calendar.opentimestamps.org
  • finney.calendar.eternitywall.com
  • ots.btc.catallaxy.com

2. Si no responde: verifica la URL con el proveedor CNE.
3. Actualiza la URL en el popup → "Guardar cambio" → se guarda automáticamente.
4. Si es un fallo temporal, el circuit breaker se recupera automáticamente. 1. Click the point on the map → "Test now" to check if it responds.

3. Update the URL in the popup → "Save change" → it saves automatically.
4. If it\u2019s a temporary failure, the circuit breaker recovers automatically.
